# DragGAN 背景保持优化与泛化性研究
本仓库包含两个独立的图像编辑研究项目，分别位于 BG_preservation 和 DragDiff_comparision 文件夹中。

## BG_preservation
基于 DragGAN 的背景保持优化研究。该项目增强了 DragGAN 在图像编辑过程中的背景保持能力，提供了交互式可视化界面，支持参数化实验配置和自定义数据集创建。

主要功能：

- 交互式拖拽编辑界面
- 背景保持程度精细控制
- 自定义实验数据集创建
- 参数化批量实验运行

## DragDiff_comparision
该项目包含与 DragDiffusion 方法的对比实验代码、数据集及评估脚本，用于比较两种图像编辑方法在不同场景下的性能。

```
两个项目分别从不同角度研究基于拖拽的图像编辑方法：

BG_preservation：专注于 DragGAN 方法的背景保持能力优化

DragDiff_comparision：系统比较 DragGAN 与 DragDiffusion 两种方法的性能差异

每个项目均有独立的 README 文档，包含详细的环境配置、使用方法和实验说明。请进入相应文件夹查看具体信息。
```
